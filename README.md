# Telegram Bot для КлассОнлайн 
 
## Установка 
 
1. Установите зависимости: 
\`\`\`bash 
pip install -r requirements.txt 
\`\`\` 
 
2. Запустите бота: 
\`\`\`bash 
python main.py 
\`\`\` 
 
## Развертывание на Railway 
 
1. Загрузите файлы на GitHub 
2. Создайте проект на Railway.app 
3. Добавьте переменные окружения: 
   - BOT_TOKEN 
   - ADMIN_ID
