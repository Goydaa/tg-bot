Telegram-бот для онлайн-школы «КлассОнлайн»
Описание проекта

Telegram-бот для автоматизации обработки заявок пользователей онлайн-школы.

Проект реализует систему записи на занятия, обработки пользовательских обращений и автоматических напоминаний.

Разработан в рамках производственной практики с целью освоения backend-разработки, архитектуры и работы с Telegram Bot API.

Используемые технологии

Python 3.11

aiogram 3.x

SQLite

Telegram Bot API

Git / GitHub

Назначение файлов
bot.py

Основной файл проекта

Содержит логику обработки сообщений, FSM-диалоги, клавиатуры и взаимодействие с базой данных.

admin_panel.py

Административный модуль

Реализует команды администратора, inline-кнопки и управление заявками.

database.py

Работа с базой данных

Создание таблиц, добавление заявок, обновление статусов и получение статистики.

utils.py

Вспомогательные функции

Генерация дат, времени и проверка данных.

main.py

Точка входа

Запускает Telegram-бот и инициализирует приложение.

Переменные окружения

Файл .env создаётся на основе .env.example

BOT_TOKEN=telegram_bot_token
ADMIN_ID=telegram_admin_id

Установка и запуск
Клонирование репозитория
git clone https://github.com/Goydaa/tg-bot.git
cd tg-bot

Установка зависимостей
pip install -r requirements.txt

Запуск
python main.py

Развертывание
