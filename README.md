# TELEGRAM-БОТ ДЛЯ ОНЛАЙН-ШКОЛЫ «КЛАССОНЛАЙН»

---

## **ОПИСАНИЕ ПРОЕКТА**

**Telegram-бот**, предназначенный для автоматизации приёма и обработки заявок пользователей онлайн-школы.

Бот позволяет:
- записываться на занятия;
- задавать вопросы;
- оставлять обращения;


Проект разработан **в рамках производственной практики** и направлен на получение практических навыков backend-разработки и работы с Telegram Bot API.

---

## **НАЗНАЧЕНИЕ ФАЙЛОВ**

### **bot.py**
**Основной файл проекта.**  
Содержит логику обработки сообщений, FSM-диалоги, клавиатуры и взаимодействие с базой данных.

---

### **admin_panel.py**
**Административный модуль.**  
Реализует команды администратора, inline-кнопки и управление заявками.

---

### **database.py**
**Модуль работы с базой данных SQLite.**  
Отвечает за создание таблиц, добавление заявок, обновление статусов и получение статистики.

---

### **utils.py**
**Вспомогательные функции.**  
Генерация дат, времени и проверка корректности данных.

---

### **main.py**
**Точка входа в приложение.**  
Запускает Telegram-бот и инициализирует приложение.

---

## **БАЗА ДАННЫХ**

В проекте используется **SQLite**, так как:
- не требует отдельного сервера;
- проста в настройке;
- подходит для небольших проектов.

Хранит:
- данные пользователей;
- заявки;
- даты и время занятий;
- статус обработки заявок.
